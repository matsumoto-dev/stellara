# Roadmap

> フェーズ定義・完了基準・依存関係。タスクの現在状態は `STATE.md` を参照。

---

## フェーズ概要

```
Phase 0 (Week 1-2)    → 戦略ドキュメント + ブランド基盤
Phase 1 (Week 2-5)    → Web App MVP（英語版・デイリーホロスコープ + Pro鑑定）
Phase 2 (Week 5-8)    → Pro機能拡張 + 決済 + マーケティング
Phase 3 (M3以降)      → LINE Bot（日本市場）
Phase 4 (M6以降)      → 最適化・スケール・新機能
```

開始日基準: オーナーのブランド名決定日 = Week 1 Day 1

> **集中戦略**: Phase 1 で英語版Web App MVPを最速リリース。日本市場（LINE Bot）は英語版の検証後に着手。決済は Phase 2 で導入（まず無料で使ってもらい、体験の質を検証する）。

## Phase 4: 最適化 + スケール

**前提:** Phase 2-3 いずれか完了

**機能ロードマップ:**

| 時期 | 機能 | 期待効果 |
|------|------|---------|
| M4 | 年額プラン | LTV向上・解約率低下 |
| M5 | 相性診断のシェア機能 | バイラル獲得 |
| M6 | ギフトカード | 新規獲得 + 単発収益 |
| M8 | 出生図（バースチャート）の詳細表示 | Pro の付加価値向上 |
| M9 | API 提供（B2B） | 他アプリへの占い機能OEM。月$99-499 |
| M10 | プレミアムプラン（$19.99） | 深い対話セッション。ARPU向上 |
| M12 | モバイルアプリ（PWA or React Native） | リテンション向上 |

**マーケティング自動化:**
- SEO ブログ記事: 週1-2本（「今週の星座運勢」等、Claude自動生成）
- YouTube Shorts: 週3本（占いコンテンツ、自動生成パイプライン）
- Reddit / Quora: 占い関連の質問にブランド認知を絡めて回答
